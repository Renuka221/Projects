# Transforming Customer Relationship Management with AI in E-Commerce

## Project Overview
This project is an AI-powered Customer Relationship Management (CRM) system for e-commerce platforms. It leverages Machine Learning techniques to analyze customer reviews, predict customer behavior, and provide personalized recommendations that improve customer satisfaction and business growth.

## Features
- User Registration and Login
- Admin Approval System
- Customer Review Analysis
- AI-Based Sentiment Classification
- Customer Behavior Prediction
- Personalized Product Recommendations
- Secure Data Management

## Technologies Used
- Python
- Django
- SQLite
- Scikit-learn
- Pandas
- NumPy
- TF-IDF Vectorizer
- Visual Studio Code

## Machine Learning
The project uses the **Naive Bayes Classification Algorithm** with **TF-IDF Vectorizer** to classify customer reviews and predict customer sentiment.

## Modules
- User Module
- Admin Module
- Owner Module
- AI Prediction Module
- Database Module

## System Workflow
1. User registers and logs in.
2. Admin approves the user.
3. Customer reviews are uploaded.
4. TF-IDF converts text into numerical features.
5. Naive Bayes predicts customer sentiment.
6. Results are displayed and stored in the database.
